# Dashboard Questions

## Q1: What is the trend of CO2 emissions over the years?
Just looking to understand if the CO2 emissions from the shipping industry are increasing or decreasing over time. This can be visualized using a line chart showing the total CO2 emissions for each reporting year.

## Q2: Which ship types are the largest contributors to CO2 emissions?
I want to identify which types of ships (e.g., container ships, bulk carriers, tankers) are responsible for the highest CO2 emissions. This can be visualized using a bar chart that breaks down total CO2 emissions by ship type.

## Q3: What is the distribution of ships by emission levels?
I want to understand how many ships fall into different emission categories (e.g., low, medium, high) based on their reported CO2 emissions. This can be visualized using a histogram

## Q4: How does fuel consumption correlate with CO2 emissions?
I want to analyze the relationship between fuel consumption and CO2 emissions to see if higher fuel consumption directly correlates with higher emissions. This can be visualized using a scatter plot with fuel consumption on the x-axis and CO2 emissions on the y-axis.

## Q5: How does fuel consumption differ across ship types?
I want to compare fuel consumption across different ship types to see if certain types of ships consume more fuel than others. This can be visualized using a box plot that shows the distribution of fuel consumption for each ship type.
